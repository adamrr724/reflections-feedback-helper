# Support AI Usage Guideline Expectations

Our long-term AI goal for GitHub Support is to creatively transform how we work, boosting efficiency by offloading repetitive tasks to AI. This enables humans to focus on impactful, nuanced areas. We are currently prioritizing building strong AI usage habits. We will define SMART goals aligned with business outcomes once sufficient data is available to set meaningful objectives.

## Tracking Contributions

Engagement with CGS/Copilot for Zendesk is currently tracked via an Azure Data Explorer dashboard that managers have access to, with broader access being developed for all support staff. Official usage metrics will be available to individuals, and similar dashboards or Azure Data Explorer metrics may be used for other tools (doc-it, spaces, spark, etc.) as these become available. Please work with your managers to track/view usage for the time being.

Active engagement with AI is currently self-driven; contributors are encouraged to advocate for their engagement with managers. If formal tracking is established, details will be shared with the broader team.
